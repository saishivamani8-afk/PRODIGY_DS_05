# PRODIGY_DS_05

## Task

Analyze traffic accident data to identify patterns related to road conditions, weather, and time of day. Visualize accident hotspots and contributing factors.

## Dataset

US Accidents Dataset (March 2023)

## Objective

Perform exploratory data analysis on accident data and identify key factors contributing to road accidents.

## Analysis Performed

* Accident Distribution by Hour of Day
* Top 10 States with Highest Accident Counts
* Weather Condition Analysis

## Libraries Used

* Python
* Pandas
* Matplotlib
* Seaborn

## Output Files

* accidents_by_hour.png
* top_states.png
* weather_conditions.png

## Key Insights

* Accident frequency varies throughout the day.
* Certain states report significantly higher accident counts.
* Weather conditions play an important role in accident occurrence.
